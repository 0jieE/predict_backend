How It Works

1. Data Processing & Cleaning

Uploaded CSV files are processed using Pandas: column names are cleaned, and data types are converted to ensure valid analysis. Missing or invalid values are handled accordingly.

2. Statistical Analysis & KPI Identification

We perform initial data processing steps such as handling missing values, converting data types, and inspecting column contents to prepare the dataset for analysis. Then, we apply Ordinary Least Squares (OLS) Regression using the statsmodels library, which is both a statistical method and a simple predictive model. This regression examines the relationship between faculty performance and predictors like workload, number of preparations, and deloading status, helping us identify key performance indicators (KPIs) based on statistical significance (p-values), effect size (coefficients), and overall model fit (R-squared).
